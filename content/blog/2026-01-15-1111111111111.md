**核心字幕内容：**

如果去掉所有的道德说教和复杂的心理学术语，直达人性的最底层，其实绝大多数的痛苦本质上源于我们试图用拥有者的心态去过一种只能是体验者的人生。

说的再直白一点，我们痛苦是因为我们误以为拥有了生活中的某些东西，并且极其傲慢地认为这些东西不仅属于我，而且应该永久属于我。

举个扎心的例子，你失恋了，为什么会痛苦并且痛不欲生？不仅仅是因为你失去了一个人。如果你去旅游住了一晚五星级的酒店，第二天退房的时候，你会痛不欲生吗？不会，你反而会开心，觉得昨晚的体验真好。为什么？因为你清楚你是来体验的，从入住的那一刻起就知道会离开。

可是当你谈恋爱的时候，大脑却悄悄的把模式切换了，潜意识里觉得这个人就是我的私人财产，他的爱，他的未来，他的关注都签了卖身契给我。所以他要走的时候，你的感觉不是租赁到期，而是私有财产被抢劫了，这种产权被侵犯的愤怒和恐慌才是痛苦的真正根源。

这就是我们最大的幻觉。在这个世界上，我们连自己的身体都只是租用几十年而已。除此之外，钱、名声、父母、孩子都只是现实提供的体验品。在这趟生命的剧本里，所有的道具都标注着八个字：仅供使用，禁止带走。

但人总是贪婪的，也总是缺乏安全感。我们拼命地想

This website is a single HTML file. It simply uses the `#anchor` suffix (from [1992](http://info.cern.ch/hypertext/WWW/Addressing/Addressing.html)) and the `:target` CSS selector to show and hide pages/content.

This setup1 is databaseless, javascriptless, and buildshit-free, so you can edit your website with a text editor and upload it somewhere2 like a normal person.

## **Do it yourself:**

1.  Download `index.html` and `style.css`
    
2.  Add a `<section>` with an `id` to define a new page in `index.html`Section HTML
    
3.  And [if you want](https://john-doe.neocities.org/#if-you-want), a link to it inside `<nav>`
    
4.  [Contact](#contact)
